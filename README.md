## 📋 What is Alice?

Alice is a text-to-video model designed to create videos from written descriptions. It uses a model with 14 billion parameters to produce high-quality videos quickly. The system completes this in just four steps, using a process called rCM distillation. Alice is open-source, which means anyone can access and use it freely.

This tool is intended for users who want to turn their ideas into videos without spending a lot of time or needing advanced technical skills.
